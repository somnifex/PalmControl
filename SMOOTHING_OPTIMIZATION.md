# 鼠标移动流畅性优化

## 优化内容

### 1. 平滑移动算法

- 使用位置历史和加权平均进行平滑处理
- 线性插值实现流畅的移动过渡
- 可调节平滑因子 (0.1-1.0)

### 2. 帧率控制

- 可设置最大移动FPS (30-240)
- 智能跳帧避免性能问题
- 精确的时间间隔控制

### 3. 性能优化

- 禁用pyautogui默认延迟
- 针对macOS系统的特殊优化
- 高效的数据结构管理

### 4. 用户界面

- 高级设置中新增平滑度控制滑块
- 最大FPS设置滑块
- 实时生效，无需重启

## 新增配置参数

```json
{
    "smoothing_factor": 0.3,
    "max_fps": 120
}
```

## 使用建议

- **平滑因子 0.3**: 推荐的平衡设置
- **最大FPS 120**: 推荐的性能设置
- 可根据个人喜好和系统性能调整
